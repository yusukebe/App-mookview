# NAME

App::mookview - View Markdown texts as a "Mook-Book" style

# SYNOPSIS

    mookview [plackup options] markdown_text.md

# DESCRIPTION

App::mookview is Plack/PSGI application for viewing Markdown texts as a "Mook-book".
